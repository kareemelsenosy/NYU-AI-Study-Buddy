IMPORTANT NOTES
===========================================

1. Set ALL variables for ALL 3 environments:

2. Exact spelling (case-sensitive):
   - PORTKEY_API_KEY (all uppercase)
   - AI_MODEL (all uppercase)
   - Files_READ_WRITE_TOKEN (exact case)

3. No spaces around = sign:
   - PORTKEY_API_KEY=value ✅
   - PORTKEY_API_KEY = value ❌

   - AI_MODEL=@vertexai/gemini-2.5-pro ✅
   - AI_MODEL="@vertexai/gemini-2.5-pro" ❌

5. After setting variables:
   - Go to Deployments → Latest → "..." → Redeploy
   - Turn OFF "Use existing Build Cache"
   - Click Redeploy
   - Wait 2-3 minutes

6. NYU Gateway Access:
   - If you get "fetch failed" errors, the gateway might require VPN
   - Contact NYU IT to whitelist Vercel IPs if needed
   - The code will automatically try direct fetch if SDK fails
